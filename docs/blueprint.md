# **App Name**: Nalanda

## Core Features:

- User Authentication & Roles: Secure authentication via Google OAuth (Gmail) with Admin, Teacher, and Student roles.
- Interactive Dashboard: Personalized dashboards with course navigation, user management, and progress tracking.
- Numerical Question Builder: Step-by-step workflow to create complex math problems with metadata and validation.
- AI Grading System: Utilize Groq AI rubrics to assess problem-solving skills, including problem understanding, formula selection, and calculation accuracy. The LLM acts as a tool which incorporates pre-configured criteria to analyze students' solutions, providing scores and pinpointing specific areas for feedback based on configured checkboxes.
- Custom Worksheet Generator: E-commerce style interface for creating assignments with a cart system and blueprint overview.
- Student Progress Tracking: Detailed stats including academic health, strengths & weaknesses analysis, and transaction history.
- Virtual Economy: Define exchange rates for coins, gold, and diamonds to incentivize student learning.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) for a sense of trust, intelligence, and focus, reflecting the academic nature of the platform.
- Background color: Light indigo (#E8EAF6) to create a calm and focused learning environment, ensuring comfortable contrast against the primary.
- Accent color: Violet (#7952B3) used for highlighting key interactive elements like CTAs, adding depth and visual interest while harmonizing with the indigo tones.
- Headline font: 'Space Grotesk', sans-serif, for headlines and shorter body texts; for its computerized, techy, scientific feel.
- Body font: 'Inter', sans-serif, to pair with Space Grotesk; suitable for longer body text.
- Consistent and clear icons to represent different subjects, units, and actions.
- Clean and intuitive layout with a collapsible sidebar for easy navigation.
- Subtle animations for rewards, progress updates, and transitions to provide positive feedback.
